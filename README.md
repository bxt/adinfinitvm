Infinite Loop Game
==================

Developing
----------

First install coffeebar:

    npm install -g coffeebar

Then run:

    coffeebar -wo script.min.js \
      src/Square.coffee \
      src/SquareDesigns.coffee \
      src/SquareView.coffee \
      src/Grid.coffee \
      src/GridView.coffee \
      src/Levels.coffee \
      src/Instructions.coffee \
      src/Zoo.coffee
